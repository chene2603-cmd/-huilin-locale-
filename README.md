# -huilin-locale-

一个轻量、可扩展的多语言前端 MVP，基于 **Next.js + React + i18next**，专为“打补丁式”新增语言设计。

## ✨ 特性

- ✅ 支持动态语言切换（URL / localStorage / 浏览器语言）
- ✅ 翻译资源按需加载（JSON 格式，命名空间组织）
- ✅ 完整示例：插值、复数、日期/货币格式化、嵌套 key
- ✅ SSR 支持：服务端渲染 + 多语言 `<title>` / `<meta>`
- ✅ 字体优化：自动加载中/日/拉丁字体
- ✅ 架构清晰：新增语言只需加 JSON + 注册一行代码

## 🚀 快速开始

```bash
npm install
npm run dev
